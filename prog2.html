What is CSS?

CSS (Cascading Style Sheets) is used to style and design webpages. It controls colors, fonts, spacing, and layout to make a website look attractive.

Why use Internal CSS?

Internal CSS is written inside the HTML file using the <style> tag. It is useful for styling a single page without creating an extra CSS file.
